# search-tool
 tool for gathering info.

How use search tool:
```bash
	pip install -r requirements
	python3 search.py
	# or
	chmod +x search.py
	./search.py
```

### Example:

```bash
	# the query is: 'oreilly hack google filetype:pdf'
>_ 	python3 search.py
   	Ingrese su busqueda: oreilly hack google filetype:pdf
```
